확장 프로그램 설치

스토어 배포 전에는 아래 수동 설치 절차로 바로 사용할 수 있습니다.

Chrome Web Store (준비 중) 수동 설치 가이드

수동 설치 (README 기준)

  1. npm run build 실행
  2. chrome://extensions 접속
  3. 우상단 개발자 모드 ON
  4. 압축해제된 확장 프로그램 로드 -> dist 폴더 선택

설치 후 기본 확인

  1. 일반 웹페이지에서 텍스트를 드래그
  2. 팝오버에서 저장
  3. 확장 UI Dashboard에서 저장 항목 확인

Notion 연동 시작 (옵션)

Settings에서 Integration Token과 Database ID를 입력하고 Auto Sync 또는 Sync Now를 사용합니다.
