CatchIt 개인정보처리방침

시행일: 2026-02-28

CatchIt(이하 "서비스")는 Chrome 확장 프로그램을 통해 웹 텍스트 저장 및 Notion 동기화 기능을 제공합니다. 본 개인정보처리방침은 서비스가 어떤 정보를 처리하는지와 그 목적을 설명합니다.

1. 수집 및 처리하는 정보

  • 사용자가 입력한 Notion Integration Token
  • 사용자가 입력한 Notion Database ID
  • 사용자가 저장한 하이라이트 데이터(텍스트, URL, 제목, 태그, 저장 시각)

2. 정보 처리 목적

  • 웹 텍스트 저장 기능 제공
  • 사용자 요청 시 Notion 데이터베이스 동기화 기능 제공

3. 정보 저장 위치 및 전송

  • 하이라이트 및 설정 데이터는 사용자 브라우저의 chrome.storage.local에 저장됩니다.
  • Notion 동기화 기능 사용 시에만 Notion API(https://api.notion.com)로 요청이 전송됩니다.

4. 제3자 제공

서비스는 사용자가 직접 실행한 Notion 동기화 요청을 제외하고 개인정보를 제3자에게 제공하지 않습니다.

5. 보관 및 삭제

  • 사용자는 확장 프로그램 내 기능을 통해 저장된 하이라이트를 직접 삭제할 수 있습니다.
  • 확장 프로그램 삭제 시 로컬 저장 데이터는 브라우저 정책에 따라 제거될 수 있습니다.

6. 보안

서비스는 기능 수행에 필요한 최소 범위의 정보만 처리하며, 사용자가 입력한 Notion 인증 정보는 서비스 외부 서버로 별도 전송하거나 저장하지 않습니다.
